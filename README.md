## ДЗ
Разработать интерактивный веб-калькулятор с расширенными аналитическими функциями, используя **HTML** для структуры, **CSS** для стилизации и **JavaScript** для логики вычислений и управления историей операций.

### 1. Базовая структура (HTML)
```html
<!-- Обязательные элементы -->
- Поле ввода/вывода результатов
- Цифровые кнопки (0-9)
- Кнопки операций (+, -, *, /, =)
- Кнопка очистки (C/AC)
- Блок для отображения истории операций
- Кнопка очистки истории
```

### 2. Стилизация (CSS)
```css
Различные цвета для разных типов кнопок
```
